# Product Spec

## Product name
Studio Macnas Unified Platform

## Problem statement
Studio Macnas currently needs one integrated system to manage bespoke bag projects, clients, materials, labor time, project costs, sustainability storytelling, and public showcase pages. The current pain is fragmentation across messaging, spreadsheets, calendars, folders, and website tooling.

## Product goal
Provide a single application that:
- tracks client and project lifecycle
- captures materials and labor
- calculates true project cost and margin
- calculates a sustainability profile
- publishes completed projects as public digital product passports
- upgrades the website into the same application

## Users
### Primary admin user
Studio owner/operator
Needs:
- create and manage clients
- create and manage projects
- log time
- sync calendar
- manage materials
- compute cost and margin
- configure sustainability axes and thresholds
- publish project showcases
- edit website content

### Public visitor
Potential customer or interested visitor
Needs:
- browse website
- view completed project showcase pages
- view story, materials, sustainability profile, and product tier
- not see internal costs or internal notes

## Product principles
- one system, one login, one data model
- deterministic calculations
- low maintenance
- mobile responsive
- strong public presentation
- private data must stay private
